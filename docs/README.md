# LLM Notify — Marketing Site (GitHub Pages)

This folder powers the public site on GitHub Pages.

## Publish

1) Commit `docs/` to the repo.
2) In GitHub → Settings → Pages:
   - Branch: `main`
   - Folder: `/docs`

Your site will be live at:
```
https://adorobantu2008.github.io/llm-notify/
```

## Downloads

The download buttons link to GitHub Releases:
```
https://github.com/adorobantu2008/llm-notify/releases/latest/download/<asset>
```

Publish new releases by uploading the ZIPs to GitHub Releases.
